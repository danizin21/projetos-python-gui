# Calculadora de IMC em Python

Aplicação gráfica simples desenvolvida em Python utilizando Tkinter para calcular o IMC (Índice de Massa Corporal).

#  Objetivo

Projeto criado para praticar:
- uso de interfaces gráficas com Tkinter
- manipulação de entradas do usuário
- criação de funções em Python

# Como funciona

O usuário informa:
- peso em quilogramas
- altura em metros

O programa calcula o IMC usando a fórmula:

IMC = peso / (altura²)

Em seguida o resultado aparece na tela.
